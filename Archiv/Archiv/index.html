HTML

HTML (Hypertext Markup Language) ist die Grundstruktur einer Website. Es definiert den Inhalt und die Struktur – zum Beispiel Texte, Bilder oder Links.

Mit HTML kannst du festlegen, was auf einer Seite zu sehen ist. Überschriften, Absätze, Listen, Bilder und Links werden alle durch HTML-Tags definiert.

CSS

CSS (Cascading Style Sheets) bestimmt das Aussehen deiner Website. Hier legst du Farben, Schriften, Abstände und Layouts fest. Ohne CSS wäre eine Seite nur schwarzer Text auf weißem Hintergrund.

CSS sorgt dafür, dass deine Seite ansprechend aussieht und auf allen Geräten gut funktioniert.

JavaScript

JavaScript macht deine Website interaktiv. Damit kannst du zum Beispiel Animationen erstellen, Formulare prüfen oder Inhalte dynamisch verändern.

Es ist die Programmiersprache, die im Hintergrund läuft und dafür sorgt, dass Webseiten lebendig werden.
